## Why does Haskell's foldl break laziness?

```haskell
foldl : Traversable a => (b -> a -> a) -> b -> Traversable a -> b
foldr : Traversable a => (b -> a -> a) -> b -> Traversable a -> b
```

```haskell
foldl (-) 0 [1, 2, 3]
-- (((0 - 1) - 2) - 3)
-- -6
foldr (-) 0 [1, 2, 3]
-- (1 - (2 - (3 - 0)))
-- (1 - (2 - 3))
-- (1 - (-1))
-- 2
```
