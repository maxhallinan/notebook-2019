Guston syntax sketch:

```lisp
(= empty?
  (fn (x)
    (if
      ((== '() x) 't)
      (else '()))))

(= rember
  (fn (x xs)
    (if
      ((empty? xs) xs)
      (if
        ((== x (first xs)) (rest xs))
        (else
          (::
            (first xs)
            (rember x (rest xs))))))))
```
